# VirtualBox Setup

## Lab Purpose
This project creates a controlled SOC home lab environment for learning security monitoring, reconnaissance, and service enumeration.

## Virtual Machines

### Analyst VM
- Operating System: Ubuntu
- Role: Security analysis and scanning

### Target VM
- Operating System: Ubuntu Server
- Role: Simulated server asset

## Virtualization
Platform:
- VirtualBox

## Network Design
The lab uses a Host-only Adapter network to keep communication isolated between virtual machines.
