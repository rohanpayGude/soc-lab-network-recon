# Ubuntu Server Installation

## Target Machine

The target system was created to simulate a server asset inside a SOC lab environment.

## Configuration

Operating System:
- Ubuntu Server

Virtualization:
- VirtualBox

Network:
- Host-only Adapter

## Installation Choices

Installation type:
- Interactive installation

Disk:
- Virtual machine virtual disk only

Encryption:
- Disabled for lab simplicity

## User Configuration

A dedicated lab user was created for administration and testing.

## Purpose

The target machine acts as an internal asset that can be discovered and analyzed by the Analyst VM.
